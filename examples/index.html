Tabulator Examples

A curated, self-contained gallery of escalating complexity — also used to smoke-test a build before release.

Before you start: these pages load the locally-built library from ../dist/, not the npm package. From the repo root run npm run build first, then open this file (works straight off file://, or serve the folder with any static server).

Each example builds on the last:

  1. Basic Table — array data, columns, fit-to-width layout, click-to-sort.
  2. Cell Formatters — progress bars, stars, ticks, money, colours and links.
  3. Sorting & Filtering — multi-sort, header filters, and programmatic filtering.
  4. Editing & Validation — input/number/list/star/date editors with validators and an edit log.
  5. Pagination — client-side paging, page-size selector and row counter over 200 rows.
  6. Grouping & Calculations — group by field with per-group & total calc rows.
  7. Row Selection — checkbox column, select-all, and live selection readout.
  8. Frozen & Responsive — frozen, movable, resizable and collapsing columns.
  9. Tree Data — nested _children, expand/collapse, tree-wide totals.
  10. Full Demo — the kitchen-sink: editing, grouping, calcs, selection, add/delete, download & persistence.
The only third-party asset on these pages is Pico.css (a classless stylesheet, loaded from a CDN) used purely for page styling. Tabulator itself stays dependency-free.
